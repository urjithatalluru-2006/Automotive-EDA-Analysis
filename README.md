# Automotive EDA Analysis

This project performs Exploratory Data Analysis (EDA) on an automotive dataset using Python.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Analysis Performed

* Data Exploration
* Statistical Summary
* Missing Value Analysis
* Histogram Visualization
* Scatter Plot Analysis
* Box Plot Analysis
* Correlation Heatmap

## Key Findings

* Higher horsepower is associated with lower fuel efficiency.
* Most vehicles fall within a moderate MPG range.
* The dataset contains a small number of missing values.
* Strong correlations exist between vehicle characteristics such as weight and displacement.
